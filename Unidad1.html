Términos asociados

Principales términos asociados a los procesos estocásticos


Experimento

Un experimento es un procedimiento llevado a cabo para apoyar, refutar, o validar una hipótesis. Los experimentos proporcionan idea sobre causa-y-efecto por la demostración del resultado, qué ocurre cuándo un factor particular es manipulado. Los experimentos varían mucho en objetivo y escala, pero se apoyan en la repetición de procedimientos y análisis lógico de los resultados.


Resultado

El resultado es el corolario, la consecuencia o el fruto de una determinada situación o de un proceso. El concepto se emplea de distintas maneras de acuerdo al contexto.

Da click en el dado para dar el resultado de sumar _ + _ = __.


Evento

Los eventos son aquellos fenómenos que surgen de ocasiones no rutinarias y que tienen objetivos de ocio, culturales, personales u organizativos establecidos de forma separada a la actividad normal diaria, cuya finalidad es ilustrar, celebrar, entretener o generar experiencias en un grupo de personas.

Obtener una cara al lanzar una moneda es un evento.


Espacio muestral

El espacio muestral es el conjunto de todos los posibles resultados de un experimento aleatorio y se suele representar como E (o bien como omega, Ω, del alfabeto griego).

Cuando lanzamos una moneda, ¿cuáles son todos los posibles resultados que podemos obtener? Que salga cara o cruz, ¿verdad? En total son dos posibles resultados, por lo que el espacio muestral tiene 2 elementos. E = {cara, cruz}


Variable discreta

Una variable discreta es aquella que está en condiciones de adoptar valores de un conjunto numérico dado. Es decir: sólo adquiere valores de un conjunto, no cualquier valor.
Entre los valores potencialmente observables de una variable discreta existe una distancia que resulta imposible de “completar” con valores intermedios. Por lo tanto, entre dos valores hay, al menos, un valor que no es observable.

Ejemplo: El número de automóviles que tiene una persona es una variable discreta. Un hombre puede tener, por ejemplo, un automóvil, dos automóviles o tres automóviles, por citar algunas posibilidades. Pero no puede tener 1,6 automóviles ni 2,8 automóviles.


Variables continuas

Una variable continua es aquella que puede adoptar cualquier valor en el marco de un intervalo que ya está predeterminado. Entre dos de los valores, siempre puede existir otro valor intermedio, susceptible de ser tomado como valor por la variable continua.

Ejemplo: Temperaturas registradas en un observatorio; tiempo en recorrer una distancia en una carrera; contenido de alcohol en un cuba-libre; estatura; tiempo de discurso de un político en las cortes insultando a los del partido contrario.


Fenómenos aleatorios

Un fenómeno aleatorio es un evento cuyo resultado, bajo situaciones similares, no se puede predecir.
Ejemplo: Crecimiento económico.


Distribución de probabilidad

La distribución de probabilidad es una herramienta fundamental para la prospectiva, puesto que con ella es posible diseñar un escenario de acontecimientos futuros considerando las tendencias actuales de diversos fenómenos.

Procesos estocásticos

Principales procesos estocásticos

Procesos IID

Los procesos IID son una clase fundamental de procesos estocásticos. Se caracterizan por dos propiedades:

Independencia: Las variables aleatorias en un proceso IID son independientes entre sí. Esto significa que el valor de una variable aleatoria no tiene ningún efecto sobre el valor de otra.

Idéntica distribución: Las variables aleatorias en un proceso IID tienen la misma distribución de probabilidad. Esto significa que la probabilidad de que una variable aleatoria tome un valor específico es la misma para todas las variables aleatorias en el proceso.

Ejemplos:

  • Lanzamiento de una moneda

    Cada lanzamiento de una moneda es un proceso IID. La probabilidad de obtener cara o cruz es la misma en cada lanzamiento, y los lanzamientos son independientes entre sí.

  • Tiros de un dado

    Cada tiro de un dado es un proceso IID. La probabilidad de obtener cada número del 1 al 6 es la misma en cada tiro, y los tiros son independientes entre sí.

Ruido Blanco

El ruido blanco es un proceso estocástico con las siguientes características:

  • Valores aleatorios

    Las muestras del proceso en diferentes puntos temporales son independientes entre sí.

  • Media constante

    La media del proceso es constante en el tiempo.

  • Varianza constante

    La varianza del proceso es constante en el tiempo.

  • Autocorrelación nula

    La autocorrelación del proceso es nula para cualquier retardo (excepto el retardo cero).

Analogía

Imagina que escuchas estática en una radio sin sintonizar ninguna estación. El sonido aleatorio de la estática es similar al ruido blanco: cada instante es independiente, con una intensidad promedio constante (ni muy fuerte ni muy débil) y sin patrón repetitivo en el tiempo.

Proceso Gaussiano

Imagina una campana de Gauss, también conocida como distribución normal. El proceso gaussiano es como una "alfombra" de estas campanas de Gauss, donde cada punto en el espacio tiene su propia campana con una altura y anchura específicas.

Características

  • Distribución normal

    Cualquier conjunto finito de variables del proceso gaussiano tiene una distribución normal multivariante.

  • Media y covarianza

    La media y la covarianza del proceso gaussiano definen la forma y la ubicación de las campanas de Gauss en la "alfombra".

  • Continuidad

    El proceso gaussiano es continuo, lo que significa que las variables del proceso cambian suavemente de un punto a otro.

Analogía

Imagina un terreno montañoso. La altura de cada punto del terreno se puede modelar como una variable aleatoria que sigue una distribución normal. Un proceso gaussiano nos permite describir la probabilidad de encontrar cualquier altura en cualquier punto del terreno. Es como tener un mapa detallado de las elevaciones, pero con la incertidumbre natural del terreno montañoso

Proceso de Poisson

Imagina que la lluvia cae de forma aleatoria en un área. El proceso de Poisson es como contar el número de gotas de lluvia que caen en un punto específico durante un intervalo de tiempo determinado.

Características

  • Llegadas independientes

    Las llegadas de eventos en un proceso de Poisson son independientes entre sí. La llegada de un evento no afecta la probabilidad de que otro evento ocurra en el futuro.

  • Tasa constante

    La tasa de llegada de eventos en un proceso de Poisson es constante en el tiempo. En otras palabras, la probabilidad de que un evento ocurra en un intervalo de tiempo pequeño es proporcional a la longitud del intervalo.

  • Sin memoria

    El proceso de Poisson no tiene memoria. La probabilidad de que un evento ocurra en un intervalo de tiempo determinado no depende de lo que haya sucedido en el pasado.

Analogía

Imagina un techo plano durante una lluvia. Las gotas de lluvia que caen sobre el techo se pueden modelar como eventos aleatorios que ocurren en el tiempo. El proceso de Poisson nos permite describir la probabilidad de que caiga un cierto número de gotas de lluvia en un intervalo de tiempo específico. Es como contar las gotas de lluvia en diferentes momentos, pero teniendo en cuenta la naturaleza aleatoria de la lluvia.

Aplicaciones

En este apartado se hablará sobre las aplicaciones de algunos de los procesos antes mencionandos en el anterior apartado.

Ruido blanco

Se puede usar para modelar errores en modelos de regresión o para detectar cambios en la dinámica de un sistema.

  • Análisis de series temporales

    Se puede usar para modelar errores en modelos de regresión o para detectar cambios en la dinámica de un sistema.

  • Simulación de Monte Carlo

    Se puede usar para generar números aleatorios para simulaciones de sistemas complejos.

  • Ingeniería de telecomunicaciones

    Se puede usar para modelar el ruido en sistemas de comunicación.

Proceso Gaussiano

  • Finanzas

    El proceso Gaussiano se utiliza para modelar el precio de las acciones y otros activos finacieros.

  • Ingeniería

    El proceso gaussiano se utiliza para modelar señales en sistemas de control y comunicaciones.

  • Aprendizaje automático

    El proceso gaussiano se utiliza en algoritmos de aprendizaje automático para realizar tareas como la clasificación y la regresión.

Proceso de Poisson

  • Telecomunicaciones

    El proceso de Poisson se utiliza para modelar el tráfico en redes de telecomunicaciones.

  • Finanzas

    El proceso de Poisson se utiliza para modelar el número de transacciones en un mercado financiero.

  • Seguros

    El proceso de Poisson se utiliza para modelar el número de reclamaciones de seguros que se presentan en un período de tiempo determinado.

Distribuciones de probabilidad

Discretas

Uniforme Discreta

La distribución uniforme discreta es una distribución de probabilidad que se utiliza para modelar situaciones en las que todos los valores posibles de una variable aleatoria tienen la misma probabilidad de ocurrir. Es decir, es una distribución de probabilidad que se aplica a variables discretas (que toman un número finito de valores) y en la que cada valor tiene la misma probabilidad de ser seleccionado.
